## Current State

The dashboard is the most-used screen тАФ it's a beautiful bento grid / activity rings layout that looks production-ready. But the data inside it is almost entirely fake seed values baked into `_initToday()`.

**What works:** Water tracker (real persistence), quick actions navigation, animated rings and bento cards, theme toggle.

**What's fake:** Sleep (hardcoded 450 min), steps (4500 hardcoded), streak ("12 Days"), bento grid protein/calories/calories burned (seed values), user name ("Aryan Suthar"), avatar (i.pravatar.cc placeholder).
